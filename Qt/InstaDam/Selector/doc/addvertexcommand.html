    Detailed Description

    The AddVertexCommand class provides undo and redo actions for adding a vertex to a SelectItem on a PhotoScene.

    AddVertexCommand is used to hold a reference to a SelectItem and the position of the new vertex point. Using the undo() and redo() commands the vertex can removed from and to the SelectItem repeatedly.

    See also AddCommand, MoveCommand, DeleteCommand, MoveVertexCommand, DeleteVertexCommand, ErasePointsCommand, and RotateCommand.
