    Detailed Description

    The RotateCommand class provides undo and redo actions for rotating a SelectItem on a PhotoScene.

    RotateCommand is used to hold a reference to a SelectItem and the starting and ending points of rotation. Using the undo() and redo() commands the rotation of the item can be undone and redone repeatedly.

    See also AddCommand, MoveCommand, AddVertexCommand, MoveVertexCommand, DeleteVertexCommand, ErasePointsCommand, and DeleteCommand.
