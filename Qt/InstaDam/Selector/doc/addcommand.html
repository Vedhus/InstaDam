    Detailed Description

    The AddCommand class provides undo and redo actions for adding a SelectItem to a PhotoScene.

    AddCommand is used to hold a reference to a SelectItem and the PhotoScene to which it belongs. Using the undo() and redo() commands the item can be added and removed from the scene repeatedly.

    See also DeleteCommand, MoveCommand, MoveVertexCommand, AddVertexCommand, DeleteVertexCommand, ErasePointsCommand, and RotateCommand.
