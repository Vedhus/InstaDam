    Detailed Description

    Creates a QCOlorDialog to select label colors with an initial widget size.

    Creates a QCOlorDialog to select label colors with an initial widget size. This is specifically used for the QtWebAssembly build as that infrastructure requires a specified size for all QDialogs.
