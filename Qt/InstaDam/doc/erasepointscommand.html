    Detailed Description

    The ErasePointsCommand class provides undo and redo actions for erasing points from a FreeDrawSelect object.

    ErasePointsCommand is used to hold a reference to a FreeDrawSelect and the PhotoScenes to which it and its mirror belong. Using the undo() and redo() commands the erasure can be undone and redone repeatedly.

    See also AddCommand, MoveCommand, AddVertexCommand, MoveVertexCommand, DeleteVertexCommand, DeleteCommand, and RotateCommand.
