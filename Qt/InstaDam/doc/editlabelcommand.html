    Detailed Description

    The EditLabelCommand command is used to change the Label of the selected item.

    EditLabelCommand is used to change the Label of the selected item. Using the undo() and redo() commands the item can be unedited and edited repeatedly.

    See also AddCommand, MoveCommand, AddVertexCommand, MoveVertexCommand, DeleteVertexCommand, ErasePointsCommand, and DeleteCommand.
