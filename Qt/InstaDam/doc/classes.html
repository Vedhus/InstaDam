    This is a listing of all InstaDam classes.

    AddCommand

    Undo and redo actions for adding a SelectItem to a PhotoScene

    AddVertexCommand

    Undo and redo actions for adding a vertex to a SelectItem on a PhotoScene

    BoxBasedSelector

    Base class for SelectItem subclasses that can be described with a box

    ColorDialog

    Creates a QCOlorDialog to select label colors with an initial widget size

    DeleteCommand

    Undo and redo actions for adding a SelectItem to a PhotoScene

    DeleteVertexCommand

    Undo and redo actions for deleting a vertex from a SelectItem on a PhotoScene

    EllipseSelect

    Class for annotating elliptical regions

    ErasePointsCommand

    Undo and redo actions for erasing points from a FreeDrawSelect object

    FreeDrawErase

    Class for erasing regions breated by a FreeDrawSelect

    FreeDrawSelect

    Class for annotating regions by painting them

    ImageList

    Handles list of images form the server

    InstaDam

    Defined the main window for the app

    Label

    Class for holding SelectItems

    LabelButton

    A button for seleting the Label for annotation

    LabelButtonFilter

    Button for filters

    Login

    Creates a login dialog

    MaskViewer

    A masked view of the image in the associated PhotoViewer

    MoveCommand

    Undo and redo actions for moving a SelectItem on a PhotoScene

    MoveVertexCommand

    Undo and redo actions for moving a vertex of a SelectItem on a PhotoScene

    PhotoScene

    Scene for rendering SelectItems

    PhotoViewer

    Viewer for displaying an image and drawing SelectItems on it

    PicPushButton

    A PushButton

    PolygonSelect

    Class for annotating irregular regions

    Project

    Holds project related information

    ProjectList

    A list of projects

    RectangleSelect

    Class for annotating rectangular regions

    Register

    Registers a user

    RotateCommand

    Undo and redo actions for rotating a SelectItem on a PhotoScene

    SelectItem

    Base class for all annotating items

    StartingWidget

    Starting widget for selecting user mode, either Local or Server

    fCheckBox

    Checbox widget for filterDialog

    fSlider

    Slider widget for filterDialog

    fSpinBox

    Spinbox widget for filterDialog

    filterControls

    Defines the properties of the mask and conducts the filtering operations

    filterDialog

    Defines a custom QDIalog based on the selected mask

    filterPropertiesMeta

    filterProperty

    Holds the filter properties

    newproject

    Creates a dialog for starting a new project
